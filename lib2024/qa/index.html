LIB2024- QA

  1. Qus: What are the three componets of Web?

    Ans: The 3 components of the web are Code Editors (to write the codes required to create the web pages), Code Validators (to ensure that this code is standardized and will be readable by all browsers), and Web Browsers (to translate the HTML code into a readable and interpretable web page)

  2. Qus: Define version control and give one example.

    Ans: VERSION CONTROL is a system that records all changes to a file (or files) over time. This lets us restore specific versions at a later time. One example of an application that does version control is Github.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: <title>(ex: <title>Resource Links)

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: <p>(ex: <p>I am a paragraph

  6. Qus: What are the tags required to create a navigation list in HTML?

    Ans: The tags are <ul> and <ol>. An unordered list is a list that has been bulleted whereas an ordered list is a list that has been numbered.

  7. Qus: What element is used to create images? Name its attributes.

    Ans: The tag that is used to create images is <img>. Its attributes are src (identifies how to get to an image), srcset (identifies an image list), usemap (identifies an image as a client operated map) and width (identifies what the width of the image is)
